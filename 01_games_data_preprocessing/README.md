# Предобработка данных о продажах компьютерных игр

Изучение и очистка датасета о продажах компьютерных игр на разных платформах с подготовкой среза данных для дальнейшего анализа.

## Цель
Изучить структуру данных, проверить их качество и корректность, выполнить предобработку и подготовить датасет для дальнейшего анализа.

## Данные
Датасет `new_games.csv` (11 столбцов, 16 956 строк): название, платформа, год выпуска, жанр, продажи по регионам (NA/EU/JP/Other), оценки критиков и пользователей, возрастной рейтинг ESRB.

## Что сделано
- Названия столбцов приведены к `snake_case`.
- Исправлены типы данных: `user_score`, `eu_sales`, `jp_sales` переведены в числовой тип (обработаны служебные значения `tbd`, `unknown`).
- Обработаны пропуски: строки без `name`, `genre`, `year_of_release` удалены; пропуски оценок сохранены; пропуски `rating` заменены индикатором `UNKNOWN`.
- Устранены неявные дубликаты (регистр жанров, устаревший рейтинг `K-A` → `E`) и 235 явных дубликатов.
- Сформирован срез `df_actual` за 2000–2013 гг.
- Добавлены категории оценок пользователей и критиков (низкая / средняя / высокая / нет оценки).

## Ключевые выводы
- В процессе очистки удалено 512 строк (~3% данных); после фильтрации осталось 12 781 запись.
- Среди игр с оценками преобладают средние и высокие оценки — и у пользователей, и у критиков.
- Топ платформ по числу игр за период — `PS2` и `DS` (свыше 2000 игр каждая); в лидерах в основном консоли Sony и Nintendo.
