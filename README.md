# Проекты Аналитика данных. Яндекс.Практикум

В этом репозитории собраны учебные и самостоятельные проекты по аналитике данных. Работы охватывают предобработку и исследование данных, визуализацию результатов, написание SQL-запросов и формирование аналитических выводов. Репозиторий будет пополняться новыми проектами по мере обучения и развития профессиональных навыков.
