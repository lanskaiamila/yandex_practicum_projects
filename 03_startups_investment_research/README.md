# Исследование стартапов и их финансирования

Проект второго модуля. Исследовательский анализ данных о стартапах (Crunchbase) для определения перспективных направлений инвестирования.

## Цель
Провести EDA данных о стартапах, изучить особенности финансирования компаний и определить наиболее перспективные сегменты и типы финансирования для потенциального инвестора.

## Данные
- `cb_investments.csv` (40 столбцов, 54 294 строки) — компании, рынки, объём и раунды финансирования, география, даты, суммы по типам инвестиций (seed, venture, angel и др.);
- `cb_returns.csv` — суммы возвратов по типам финансирования по годам.

## Что сделано
- Предобработка: очистка `funding_total_usd` от разделителей и индикаторов, приведение к числу; даты → `datetime`; заполнение текстовых пропусков заглушкой `unknown`; восстановление `mid_funding_at` серединой интервала; удаление дубликатов.
- Инжиниринг признаков: группы по срокам финансирования (единичное / до года / более года); деление рынков на массовые, средние (`mid`) и нишевые (`niche`).
- Работа с выбросами по методу IQR отдельно для каждого сегмента рынка.
- Анализ динамики финансирования по годам, по массовым сегментам и доли возвратов по типам финансирования.

## Ключевые выводы
- Удалено ~24,7% строк на предобработке; осталось 40 905 записей.
- Типичный объём финансирования компании — около 2 млн USD (медиана), распределение сильно смещено вправо.
- Компании с финансированием дольше года дают ~62% всех инвестиций при ~30% числа компаний.
- `venture` — лидер по объёму, распространённости и возвратам; наиболее устойчивая динамика доли возвратов.
- Растущие в 2014 сегменты: `Medical`, `Technology`, `Internet`.
- **Рекомендация:** сегмент `Medical` с венчурным финансированием (с оговоркой о неполноте данных за конец 2014).
