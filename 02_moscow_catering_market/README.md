# Исследование рынка заведений общественного питания Москвы

Исследовательский анализ рынка общепита Москвы и подготовка рекомендаций для инвесторов фонда «Shut Up and Take My Money», планирующих открыть заведение.

## Цель
Провести EDA рынка заведений общественного питания Москвы и сформировать рекомендации по выбору формата, локации и ценовой политики для нового заведения.

## Данные
Два датасета на основе Яндекс Карт и Яндекс Бизнеса (лето 2022):
- `rest_info.csv` — название, адрес, округ, категория, часы работы, рейтинг, признак сети, число посадочных мест;
- `rest_price.csv` — ценовая категория, средний чек, медиана среднего чека и цены капучино.

## Что сделано
- Объединение датасетов методом `merge()` (left) по `id`.
- Предобработка: типы данных, обработка пропусков (`hours`, `seats`, ценовые столбцы), нормализация текста, удаление неявных дубликатов.
- Инжиниринг признака `is_24_7` (круглосуточный режим).
- EDA по 8 задачам: категории заведений, распределение по округам, сетевые/несетевые, посадочные места, рейтинги, корреляция рейтинга (`phi_k`), топ-15 сетей, средний чек по районам — с визуализациями.

## Ключевые выводы
- Рынок сконцентрирован вокруг кафе, ресторанов и кофеен (~70% заведений).
- Лидер по концентрации — Центральный округ (~27%); несетевые заведения преобладают (62% против 38%).
- Рейтинги высокие (средний 4.23) и слабо различаются между категориями; сильнее всего связаны с ценовой категорией.
- Самая крупная сеть — «Шоколадница» (120 заведений).
- Средний чек выше в центральных и западных округах (~1000 ₽), ниже на окраинах (450–500 ₽).
- **Рекомендации:** формат кофейни/кафе на 60–80 мест, округа с балансом спроса и конкуренции, акцент на качестве сервиса.
